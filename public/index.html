About BookHub

BookHub is a Full Stack Web Application developed using HTML, CSS, Bootstrap, JavaScript, Node.js, Express, SQLite and Multer.

Users can:

  • Login & Register
  • Add their own books
  • Upload custom book cover images
  • Browse available books
  • Purchase books
  • View My Purchases
  • Checkout securely
